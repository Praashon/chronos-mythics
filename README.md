# Chronos Mythics

A premium digital experience.

## Overview

Chronos Mythics represents a convergence of immersive design and modern engineering. Built to deliver fluid, high-fidelity user interactions through a composition of advanced web technologies.

## Architecture

The interface is constructed with a focus on performance and visual precision.

- **Framework** — Next.js 16
- **Core** — React 19RC
- **Styling** — Tailwind CSS 4
- **Animation** — Framer Motion
- **3D Environment** — Three.js & React Three Fiber
- **State** — Zustand
- **Backend** — Supabase

## Development

To initialize the development environment:

```bash
npm run dev
```

The application will be accessible at [http://localhost:3000](http://localhost:3000).
